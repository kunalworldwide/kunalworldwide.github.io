## Overview

Azure VM Price Checker is a web application that enables quick and efficient comparison of Azure VM prices across all regions. Built with Python and Streamlit, it leverages the Azure Retail Prices API to provide instant price comparisons.

## Key Features

- Quick SKU search with real-time filtering
- Comprehensive price comparison across all Azure regions
- Support for Spot, Reserved Instances, and Dev/Test pricing
- Fast, responsive interface with concurrent API calls
- No Azure credentials required
